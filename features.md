# Features.md

## Core Features
- Real-time text chat with WebSocket
- Per-thread language translation
- 2 free additional translation languages per user
- Whisper-based voice transcription + translation
- Media and document messaging
- End-to-end encryption (Signal Protocol)
- Voice/video calls via WebRTC

## Moat Features
- Emotion detection from text/audio
- Real-time toggling between languages
- Multilingual group chat per user preference
- Inline "show original" feature

## Technical
- Scalable microservices
- Cloud-agnostic deployment
- Optimized NoSQL schema for chat history
