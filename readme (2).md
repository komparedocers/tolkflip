# README.md

## Cross-Cultural Chat Application

Enables users worldwide to communicate through multilingual real-time chat with automatic translations and transcriptions.

### Setup

1. Install Docker, Kubernetes (for prod)
2. Deploy services with `docker-compose` or Helm
3. Configure Cassandra, Redis, object storage (e.g., MinIO)
4. Launch mobile clients via Android Studio (Java) or Xcode (Swift)

### Features
- Real-time multilingual messaging
- Whisper-based voice message transcription
- Secure media sharing and video calls
- End-to-end encrypted (Signal Protocol)
- Scalable to millions of users
