# Features

## Messaging
- 1:1 and group text chat
- Message status (sent / delivered / read)
- Infinite history with lazy loading

## Translation
- Default language per user
- Per‑thread language selection
- Up to 2 extra languages per user for free
- Automatic translation of incoming messages into the chosen view language
- Toggle to show original vs translated text

## Audio & Media
- Voice notes
- Whisper‑based transcription + translation of voice notes
- Image, video, and document sharing (encrypted and stored in object storage)

## Calls
- 1:1 voice and video calls (WebRTC‑style)

## Presence & UX
- Online / offline indicators
- Typing indicators
- Dead‑simple phone‑number signup

## Security
- End‑to‑end encryption (Signal‑style)
- Transport‑level TLS
- Encrypted media

## Moat Features
- Per‑thread on‑the‑fly language switching
- Multilingual group chats (each user sees messages in their own language)
- Optional emotion/tone hints based on text/audio analysis
